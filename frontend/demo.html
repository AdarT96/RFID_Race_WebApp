📡 חיבור לבקרי ESP32

כאשר המצב פעיל — לחיצה על "התחל מרוץ" / "עצור" / "איפוס" תשלח פקודות HTTP אמיתיות לבקר התחנה המתאימה, והמערכת תקרא תגים ישירות ממנו. כל ESP32 מייצג תחנה, ומחובר לאותה רשת Wi-Fi של המכשיר עליו פועלת האפליקציה.

⚠ הדפדפן חוסם HTTP מדף HTTPS. הפעל את האפליקציה מקומית (python -m http.server / Live Server) כדי שהקריאות לבקרים יעבדו.
